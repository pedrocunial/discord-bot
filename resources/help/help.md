# HELP!!

## Comandos

Todos os comandos começam com o prefixo `>`, por exemplo, para pedir uma
 frase do pythonzeras, use `>python`.

### python / pythonzeras

Retorna um trecho de pythonzeras.

### gabs / comissao

Colação cancelada.

### play / tocar / youtube <url ou palavras chaves>

Adiciona uma música a playlist, caso não exista uma playlist ativa, cria uma
 e adiciona o bot no canal.
 
Exemplo: `>play supla nightcore`

### nightcore <palavras chave>

Alias para `>play nightcore <pavras chave>`.

### delete / remove <index>

Remove música de índice `<index>` na fila.

### queue / fila

Mostra a fila de músicas.

### skip

Pula a música atual.

### stop / clear

Limpa a playlist atual.

### goto / jump <index>

Pula para música de índice `<index>` na fila.

### thomas / tomas

thomas

### help

Você já deveria saber esse...
